# Exercise 02: Create an agent


When a customer first reaches out, you'd route them to an agent that can answer common questions before handing the conversation off to a live representative. In this exercise, you'll create a simple agent and connect it to a single knowledge source.

{: .note }
> In a real-world implementation, you would typically configure the agent with multiple articles or knowledge sources. For a customer demo with a specific talk track, however, it helps to keep the experience tightly scoped so the story is clear and predictable. In this case, the simplest approach is to connect the agent to a single article. If you prefer, you can add additional sources.
