# Exercise 06: Configure intent-based suggestions for Copilot agents (Bonus)


Now that you've worked with the application, let's look at an alternative scenario. Just like you did with turning on Intent based suggestions for representatives, you can turn on intent based suggestions for specific agents. When enabled, the agent will identify intents while the customer is interacting with the agent. They can use items like attributes and suggestions will be leveraged to assist the agent in solving the issue.
